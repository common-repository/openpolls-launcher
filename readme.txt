=== OpenPolls Launcher ===
Contributors: openpolls
Tags: openpolls, online public polls
Requires at least: 4.0
Tested up to: 4.7
Stable tag: 1.0

Easily install OpenPolls Launcher using this simple plugin.

== Installation ==
= For automatic installation: =
* Go to Plugins > Add New
* Search for OpenPolls Launcher
* Click Install Now
* If your installation is successful, click Activate
* Go to Settings > OpenPolls Launcher and paste your Launcher code, which is located under the Settings tab in your OpenPolls profile

= For manual installation: =
* If you have a zip archive, extract the file
* Upload the folder to wp-content/plugins in your WordPress directory
* Go to Plugins and find OpenPolls Launcher in the list
* Click Activate
* Go to Settings > OpenPolls Launcher and paste your Launcher code, which is located under the Settings tab in your OpenPolls profile
